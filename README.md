Weather Now

Project Overview

Weather Now is a weather dashboard application built with React that allows users to search for current weather conditions in cities around the world. The application retrieves real-time weather data from the OpenWeatherMap API and displays important information such as temperature, humidity, wind speed, and the current weather condition.

This project was developed as part of the ALX Frontend Engineering Capstone Project. The goal is to demonstrate practical frontend development skills including API integration, responsive interface design, state management, and application deployment.

Features

City Search

Users can search for any city and view its current weather conditions.

Weather Information

The dashboard displays the following information for each searched city:

* Current temperature
* Humidity level
* Wind speed
* Weather condition icon and description

Error Handling

If a user enters an invalid city name or if the API request fails, the application displays a clear error message informing the user that the city could not be found.

Responsive Design

The interface is styled using vanilla CSS and is responsive across different screen sizes including desktop, tablet, and mobile devices.

Automatic Weather Updates

The application refreshes weather data automatically at regular intervals to ensure users receive up-to-date information.

### Recent Searches

Recent city searches are stored in the browser’s local storage so users can easily access previously searched locations.


Technologies Used

The project was built using the following technologies:

* React JS
* JavaScript (ES6)
* HTML5
* CSS3
* Vite
* OpenWeatherMap API
* Git and GitHub


API Configuration

This project uses the OpenWeatherMap API.

Deployment

The application can be deployed using platforms such as Vercel or Netlify.

Once deployed, the live version of the project will be accessible through a public URL that can be shared for testing and review.
